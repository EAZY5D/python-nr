```
usage: nr archive [-h] [-e EXTRACT] [-s] filename

positional arguments:
  filename              Name of the archive.

optional arguments:
  -h, --help            show this help message and exit
  -e EXTRACT, --extract EXTRACT
                        Extract the archive to the specified directory.
  -s, --single-unpack   Skip over the top directory in the archive if it is
                        the only member in the archive root directory.
```
